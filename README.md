
## Face Recognition Attendance System

A simple Python-based face recognition attendance system using **InsightFace** and **OpenCV**.
The system recognizes registered faces through a webcam and automatically marks attendance with visual confirmation.

---

## Features

* Real-time face detection and recognition
* High-accuracy cosine similarity matching
* Attendance marked only once per session
* Visual confirmation when attendance is recorded
* Works on Apple Silicon (M-series Macs)

---

## Requirements

* Python 3.9+
* OpenCV
* InsightFace
* NumPy
* ONNX Runtime
